-o- Use docker, solvator, goat modules extensively for initial structure, conformer search. 

-o- Use xtb, gfn2xtb, gfn-ff carefully for relatively complex molecules, e.g., transition metal complexes. Check xtb or orca webpage while making that decision. 

-o- Use b97-3c and r2scan-3c methods for initial DFT calculations as these include several important correction factors by default. 

-o- Use better quality functionals/basis sets, zero-point energy correction, entropic contributions only for the final stage to save computational time.
